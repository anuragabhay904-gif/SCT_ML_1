This project implements a Multiple Linear Regression model to estimate residential real estate prices. Using Python and scikit-learn, the model analyzes the relationship between a house's market value and key features: total square footage, bedroom count, and bathroom count.

# Residential Real Estate Price Prediction

This project implements a Multiple Linear Regression model to estimate house market values based on key physical attributes.

## 🚀 Overview
The model uses Python and scikit-learn to analyze the relationship between price and:
* **Square Footage**
* **Bedroom Count**
* **Bathroom Count**
